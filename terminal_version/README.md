# Comparador FBref x Transfermarkt — Versão Terminal

Esta pasta contém a versão **para CMD/VS Code (sem IPython obrigatório)** do projeto.

## Como instalar
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Como executar
```bash
python main.py --salvar-saidas ./saidas
```
